# ARM Prestige

## Introduction

ARM Prestige is an investment platform built with Next.js (App Router). It enables `Prestige` client to invest through the ARM investment platform.

## Getting Started

Guide to getting the project up and running on your local system.

1. Installation process
2. Software dependencies
3. Latest releases
4. API references

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build and Test

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

### Lint

```bash
npm run lint
```
